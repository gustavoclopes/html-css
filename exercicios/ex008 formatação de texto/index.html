Principais de formatação

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <b> (sem semântica, ou seja, somente possui forma)

Nesta frase, temos um termo em destaque usando a tag <strong> (com semântica, ou seja, com significado, com sentido!)

Itálico / Ênfase

Nesta frase, temos um trecho em itálico usando a tag <i> (sem semântica)

Nesta frase, temos um termo em ênfase usando a tag <em>. (com semântica)


Obs.: Para envelopar basta pressionar "Ctrl+Shift+P" - Após "abb", e selecionar "Wrap with Abbreviation" e por fim digitar o comando, ex.: i


Texto marcado

Podemos criar também em texto marcado usando a tag <mark>

Podemos também modificar a cor do trecho marcado (css) <mark style="background-color: aqua;">

Texto deletado

Podemos ter um texto deletado, usando a tag <del>

Texto inserido (sublinhado)

Podemos ter um texto inserido, usando a tag <ins>

Texto sobrescrito

Para inserir coias do tipo x20+3, utilizamos a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O, utilizamos a tag <sub>
